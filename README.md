# NCU Compiler Final Project: Mini LISP
Make a Mini-Lisp interpreter with Python that has these features:
Basic:
- [ ] Syntax Validation
- [ ] Print
- [ ] Numerical Operations
- [ ] Logical Operations
- [ ] if Expression
- [ ] Variable Definition
- [ ] Function
- [ ] Named Function
Bonus:
- [ ] Recursion
- [ ] Type Checking
- [ ] Nested Function
- [ ] First-class Function

## Requirement
* python 3
* lark
